Players

Create a player

POST /players

Request:

{
  "first_name": "Igor",
  "last_name": "Shesterkin",
  "sweater_number": 31,
  "position": "G",
  "birth_date": "1995-12-30",
  "birth_country": "RUS",
  "shoots_catches": "L",
  "current_team_id": 1,
  "is_active": true
}

Response:

{
  "player": {
    "id": 35,
    "is_active": true,
    "current_team_id": 1,
    "first_name": "Igor",
    "last_name": "Shesterkin",
    "sweater_number": 31,
    "position": "G",
    "birth_date": "1995-12-30",
    "birth_country": "RUS",
    "shoots_catches": "L"
  }
}

Update a player

PATCH /player/{id}

Request:

{
  "first_name": "Connor",
  "last_name": "McDavid",
  "sweater_number": 31,
  "position": "G",
  "birth_date": "1995-12-30",
  "birth_country": "RUS",
  "shoots_catches": "L",
  "current_team_id": 1,
  "is_active": true
}

Response:

{
  "player": {
    "id": 1,
    "version": 3,
    "is_active": true,
    "current_team_id": 1,
    "first_name": "Connor",
    "last_name": "McDavid",
    "sweater_number": 31,
    "position": "G",
    "birth_date": "1995-12-30",
    "birth_country": "RUS",
    "shoots_catches": "L"
  }
}

Delete a player

DELETE /player/{id}

Example response:

{
  "message": "player successfully deleted"
}
