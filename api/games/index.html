Games

Create a game

POST /games

Request:

{
  "home_team_id": 1,
  "away_team_id": 2,
  "start_time": "2026-03-10T19:30:00-05:00"
}

Response:

{
  "game": {
    "id": 2,
    "home_team_id": 1,
    "away_team_id": 2,
    "start_time": "2026-03-10T19:30:00-05:00",
  }
}
